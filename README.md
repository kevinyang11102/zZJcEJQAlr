# 项目概述 java676

本项目是一个基于协同过滤算法的校园二手交易市场系统，采用SSM框架与Vue.js技术构建。系统提供商品信息浏览、校园资讯、留言反馈和个人中心等功能。

# 技术栈

## 前端技术
- Vue.js
- Element-Plus

## 后端技术
- SSM（Spring、SpringMVC、MyBatis）
- Mybatis-Plus

## 数据库
- MySQL

# 功能模块

## 用户端
- 首页
- 商品信息
- 校园资讯
- 留言反馈
- 个人中心

## 管理端
- 用户管理
- 商家管理
- 商品类型管理
- 商品信息管理
- 商品购买管理
- 交流信息管理
- 交流回复管理
- 留言板管理
- 系统管理

# 系统角色
- 普通用户：可以浏览商品信息，发布商品，留言反馈等。
- 商家：可以管理自己的商品，处理订单等。
- 管理员：负责整个系统的用户、商品、信息等的管理。

# 运行环境
- Java环境
- MySQL数据库
- Web服务器（如Tomcat）

# 部署步骤
1. 准备Java运行环境。
2. 安装MySQL数据库并导入数据。
3. 部署应用到Web服务器。
4. 启动应用。
